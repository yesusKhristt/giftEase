The <blockquote> tag in HTML is used to define a section of text that is quoted from another source. It typically indents the text to visually distinguish it as a block of quotation.The cite attribute in a <blockquote> tag is not visible to the user by default. Superscript and Subscript

Tables

In HTML, the <table> tag is used to create tables. Tables are used to display data in rows and columns. The <tr> tag is used to define a row in a table, and the <td> tag is used to define a cell in a row. The <th> tag is used to define a header cell in a table.
Here’s how it works:

  • <table>: Defines the entire table.
  • <tr>: Defines a row in the table.
  • <td>: Defines a cell in a row.
  • <th>: Defines a header cell in a table.

Ex:
in HTML :

Anchor Tag (<a>)

Syntax for anchor tag : <a href="URL">Link Text</a>

Ex:

Attributes of anchor tag:

When to use anchor tag:

  • To create hyperlinks for navigation within or between webpages.
  • To link to external websites, internal page sections, or download files.

NAME Attribute

Definition:

The name attribute is used to identify elements for form data submission or linking within a document. It is commonly applied to form controls like:

  • <input>
  • <textarea>
  • <button>
  • <select>

Usage: The name attribute specifies a name for the element, which is used during form submission to associate data with the corresponding element.

Difference from ID:

  • id: Uniquely identifies an element on a page and is primarily used for JavaScript or CSS.
  • name: Used for form data submission and can have multiple elements with the same value.

Examples:

1. Form control:

HTML:
<form method="post">
  <label>Name: </label>
  <input type="text" name="username">
</form>
Explanation: The name "username" associates the input data with the key "username" when the form is submitted.

2. Internal linking:

HTML:
<a name="section1"></a>
<a href="#section1">Go to Section 1</a>
Explanation: The name attribute creates a target that can be referenced by hyperlinks with href.

Note: The name attribute for the <a> tag is considered obsolete in modern HTML. Use id instead for creating link targets.

Syntax: <img src="image.jpg" alt="Description width="100px" height="100px">

Image Mapping

Image mapping is a technique in HTML that allows you to create clickable areas on an image that link to different webpages or resources. It involves defining shapes (rectangles, circles, or polygons) on an image and associating them with hyperlinks using the <map> and <area> tags.

Here's how it works:

  • <img>: Embeds the image in the webpage.
  • <map>: Defines the image map and its areas.
  • <area>: Specifies the clickable areas on the image.
    •                     Syntax:
                    <img src="image.jpg" alt="Image" usemap="#mapname">
                    <map name="mapname">
                        <area shape="rect" coords="x1,y1,x2,y2" href="page1.html" alt="Area 1">
                        <area shape="circle" coords="x,y,r" href="page2.html" alt="Area 2">
                        <area shape="poly" coords="x1,y1,x2,y2,x3,y3" href="page3.html" alt="Area 3">
                    </map>

    • shape: Specifies the shape of the clickable area (rect, circle, poly).
    • coords: Defines the coordinates of the shape (e.g., x1,y1,x2,y2 for a rectangle).
      • for a rectangle
        • x1: The x-coordinate of the upper-left corner.
        • y1: The y-coordinate of the upper-left corner.
        • x2: The x-coordinate of the lower-right corner.
        • y2: The y-coordinate of the lower-right corner.
      • for a circle
        • x: The x-coordinate of the center.
        • y: The y-coordinate of the center.
        • r: The radius of the circle.
      • for a polygon
        • x1, y1, x2, y2, x3, y3, ...: The coordinates of the polygon's vertices.
    • href: Specifies the URL of the linked page or resource.
    • alt: Provides alternative text for the clickable area.
    • usemap: Associates the image with the map by referencing the map's name.
    • name: Defines the name of the map for reference.
    • alt: Provides alternative text for the image map.

    Ex:

                        <img src="cat.jpg" alt="cute cat" usemap="#CAT">
                    <map name="#CAT">
                        <area shape="poly" coords="288,174,295,152,318,152,321,175,304,179," href="#" alt="Polygonal" title="Eye">
                        <area shape="rect" coords="249,190,279,212" href="#" alt="Clickable Nose" title="Nose">
                        <area shape="circle" coords="265,355,20" href="#" alt="tummy" title="Tummy">
                    </map>

HTML FORMS
